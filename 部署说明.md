# OpenClaw 银龄行业自动发布工作流 - 部署指南

本文件夹包含了配置好的 OpenClaw 工作区、所有的自定义技能（Skill），以及视频号自动化生成的底层 Python 脚本。请按照以下步骤在新的 Mac 上完成部署。

## 1. 文件夹归位
将 `workspace` 文件夹和 `openclaw_config_template.json` 放到新机器的 `~/.openclaw/` 目录下（如果该目录不存在，请先安装 OpenClaw 或手动创建）。
- 重命名 `openclaw_config_template.json` 为 `openclaw.json`，覆盖或替换掉系统默认的配置。
- 将打包的 `workspace` 复制并合并到新机器下的 `workspace` 目录。

## 2. 填写关键配置与 Token
为了保证数据安全，所有的 API Key 和密钥已经从该部署包中移除。您需要手动补全以下信息：

### 2.1 主配置 (`~/.openclaw/openclaw.json`)
打开并编辑新机器上的 `openclaw.json`：
1. 找到 `"gateway" -> "auth" -> "token"`，填入您的网关 Token。
2. 找到 `"models" -> "providers" -> "custom-api-n1n-ai" -> "apiKey"`，填入您的 API Key。

### 2.2 飞书凭证配置 (`~/.openclaw/credentials/lark.secrets.json`)
由于通信通道依赖飞书，请在新机器上创建目录和文件 `~/.openclaw/credentials/lark.secrets.json`，并填入您的飞书 `AppSecret`，格式如下：
```json
{
  "/lark/appSecret": "您的飞书_APP_SECRET"
}
```
同时注意 `openclaw.json` 里的 `"appId": "cli_a9521c72c2799bcf"` 对应的是您的飞书机器人 ID。

### 2.3 工作区工具配置 (`workspace/TOOLS.md`)
打开 `workspace/TOOLS.md`：
找到底部的微信公众号配置项，将其替换为真实的 AppID 和 Secret：
```bash
WECHAT_APP_ID=YOUR_WECHAT_APP_ID_HERE
WECHAT_APP_SECRET=YOUR_WECHAT_APP_SECRET_HERE
```

## 3. 初始化 Python 环境
视频生成的自动化底层依赖于 `workspace/openclaw_upload`。由于本地环境差异（如路径和架构不同），虚拟环境 `.venv` 并没有被打包，需要您重新构建。
在新机器上打开终端，执行以下命令初始化虚拟环境：
```bash
cd ~/.openclaw/workspace/openclaw_upload
python3.11 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
*(注意：请确保新机器已安装 Python 3.11 及以上版本以及必要的浏览器。)*

## 4. 启动与验证
完成以上配置后，启动 OpenClaw。
您可以在飞书上对机器人发送图片，测试触发“银龄行业”相关的图生视频与发布工作流。首次涉及发布时，系统会自动将登录二维码推送至飞书，扫码后即可长期保活自动运行！
